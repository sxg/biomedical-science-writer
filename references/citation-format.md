# Citation Formats

Reference formatting for common biomedical journal styles.

## Vancouver Style (ICMJE)

Used by: Most medical journals, Radiology, JAMA, NEJM, Lancet

### Journal Article
```
Author AA, Author BB, Author CC. Title of article. Journal Name. Year;Volume(Issue):Pages. doi:xx.xxxx/xxxxx

Smith JA, Jones BB, Wilson CC. MR fingerprinting for brain tumor characterization. Radiology. 2023;308(2):423-431. doi:10.1148/radiol.2023230456
```

### Up to 6 authors: list all
### More than 6 authors: list first 3 + "et al."
```
Smith JA, Jones BB, Wilson CC, et al. Title. Journal. Year;Vol:Pages.
```

### In-text Citation
Numbered in order of appearance: [1], [2-4], [1,5,7]

## AMA Style (American Medical Association)

Used by: JAMA network journals

### Journal Article
```
Author AA, Author BB. Title of article. Journal Name Abbrev. Year;Volume(Issue):Pages. doi:xx.xxxx/xxxxx

Smith JA, Jones BB. MR fingerprinting for brain tumor characterization. Radiology. 2023;308(2):423-431. doi:10.1148/radiol.2023230456
```

### Book
```
Author AA. Title of Book. Edition. Publisher; Year.

Haaga JR. CT and MRI of the Whole Body. 6th ed. Elsevier; 2017.
```

### In-text
Superscript numbers: findings¹ or findings.¹⁻³

## APA Style (7th Edition)

Used by: Psychology, some neuroscience journals

### Journal Article
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Journal Name, Volume(Issue), Pages. https://doi.org/xx.xxxx/xxxxx

Smith, J. A., Jones, B. B., & Wilson, C. C. (2023). MR fingerprinting for brain tumor characterization. Radiology, 308(2), 423-431. https://doi.org/10.1148/radiol.2023230456
```

### In-text
Author-date: (Smith et al., 2023) or Smith et al. (2023)

## DOI Formatting

Preferred format: doi:10.xxxx/xxxxx
Alternative: https://doi.org/10.xxxx/xxxxx

## Quick Reference: What to Include

| Element | Required | Notes |
|---------|----------|-------|
| Authors | Yes | Format per style guide |
| Title | Yes | Sentence case for Vancouver/AMA |
| Journal | Yes | Use standard abbreviation |
| Year | Yes | Publication year |
| Volume | Yes | Usually |
| Issue | Sometimes | Some journals omit |
| Pages | Yes | Or article number |
| DOI | Preferred | If available |
| PMID | Optional | Useful but not required |

## Generating References from PubMed

When using source notes, format references from:
- Title: from metadata
- Authors: from metadata (format per style)
- Journal: use abbreviation
- Year/Volume/Pages: from metadata
- DOI: from identifiers
